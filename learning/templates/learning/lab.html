Example Sentence

例句：Agent 调用 工具

我们只看 3 个 token，目标是回答：每个词在理解自己时，应该参考其他词多少？

Agent调用工具
Q = 我在找什么当前 token 发出的“查询”
K = 我有什么特征其他 token 可被匹配的“钥匙”
V = 我携带的信息最终被加权汇总的内容

Step 1

QKᵀ：计算相关性

把 Query 与每个 Key 做点积，分数越高表示当前词越应该关注那个词。

score = Q · Kᵀ
这一步输入
Q 矩阵、K 矩阵
这一步输出
每个 token 对其他 token 的原始关注分数
怎么读图
看一行：这一行的词正在决定关注谁；颜色越亮，关注越强。
例如“调用”这一行更关注“Agent”和“工具”，因为它需要知道谁在调用、调用什么。
